.. _Classes_DefinitionSyntax:

Definition Syntax
-----------------

You introduce a class with the ``class`` keyword and place its definition
within a pair of braces: 

.. testcode:: Classes
    
    -> class someClass {
           // class definition goes here
       }   

Here is an example of a class definition: 

.. testcode:: Classes

    -> class ________ {
           // to be filled in
       }
